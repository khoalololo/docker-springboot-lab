Architecture Overview

This lab demonstrates a complete microservices architecture with:

Traefik

  • Automatic service discovery
  • Path-based routing
  • Load balancing
  • SSL termination ready

Backend

  • Spring Boot 3.2
  • JPA/Hibernate
  • PostgreSQL
  • RESTful API

Security

  • Docker Secrets
  • Network isolation
  • No exposed passwords
  • Production-ready

Docker

  • Multi-stage builds
  • Docker Compose
  • Health checks
  • Auto-restart
